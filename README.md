## 3 || 3

请在包含开场在内的**三条信息内**，或者第一条信息开始计时**三分钟内**尽量详细的说明来意。

而且“3 || 3”原则，尤其后半段，更侧重于一对一的私聊会话，群聊会话对于信息要求的组织要求可能更高（如果群里不是很冷清的话。
